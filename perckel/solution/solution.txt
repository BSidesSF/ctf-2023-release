Reading the hash with Storable and using decode_flag to decode the flag:


#========= start decode_flag.pl ==========
#!/usr/bin/perl

use strict;
use warnings;

no warnings 'once';

use Storable qw(retrieve);

$Storable::Eval = 1;


my $f_ref = retrieve('flag.bin');

print $f_ref->{'decode_flag'}($f_ref->{'flag'}), "\n";
#========= end decode_flag.pl ==========


$ ./decode_flag.pl
