Search for Extra Flag Intelligence

Join the search for extra flag intelligence (SEFI)! Our custom client will download work units to process for signals and attempt to locate them. If we process enough signals, we hope to locate a flag!

We think they're using some kind of extended line code to transmit, but we haven't implemented the right one fully yet. We're starting on some implementations.
